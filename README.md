# Udemy Finance & Accounting Courses Analysis

This project analyzes Udemy course data specifically from the **Finance & Accounting** category. Using Python and data science techniques, the notebook explores course metrics such as ratings, price distribution, subscriber activity, content duration, and overall course popularity. The goal is to identify trends, insights, and patterns that help understand what makes a successful course in this category.
